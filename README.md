# pure

#### 介绍

基于SpringBoot+Vue前后端分离的Java快速开发框架，微服务

#### 端口架构

| 端口   | 说明            | 服务名           | 访问路径                         |
|------|---------------|---------------|------------------------------|
| 80   | 前端项目          | pure-ui       | http://localhost             |
| 8000 | 网关服务          | pure-gateway  | http://localhost:8000        |
| 8100 | 权限认证          | pure-auth     | http://localhost:8100/auth   |
| 8200 | 系统模块          | pure-system   | http://localhost:8200/system |
| 8300 | 代码生成          | pure-codegen  | http://localhost:8300/gen    |
| 8400 | 文件模块          | pure-file     | http://localhost:8400/file   |
| 8500 | 定时任务调度中心      | xxl-job-admin | http://localhost:8500        |
| 8600 | 定时任务执行器（示例模块） | pure-job      | http://localhost:8600        |
